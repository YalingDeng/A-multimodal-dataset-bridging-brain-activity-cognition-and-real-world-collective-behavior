Preprocessing code:
1preprocess_filter.py
Read the raw data in bdf format, filter and segment the data into resting-state and task-state (watching videos) segments, and store the processed data in a unified location.
2preprocess_check_channel.py
Read the filtered data, open the data in batches and plot the EEG signals. Manually inspect and select bad leads, which will be saved as an Excel file.
3preprocess_interpolate.py
Based on the bad lead Excel generated by 2preprocess_check_channel.py, perform batch interpolation on the problematic data, store the uninterpolated data separately, and overwrite the filtered data with the interpolated data.
4preprocess_reref_ica.py
Perform re-referencing (whole-brain averaging) on the EEG files, run the ICA algorithm, calculate the artifact components and probabilities using ICALABLE, and save the automatically identified artifacts in an Excel file for easy addition of new components.
5preprocess_repair_artifact.py
Read the artifacts from the Excel file with added electromyography components, delete the artifacts, and segment the video.

The formal processing part of the code:
6process_cut_event.py
Split each video into individual files.
7process_cal_psd.py
Calculate PSD.
8process_read_power_export.py
Save the calculated PSD in frequency bands to an Excel file.
